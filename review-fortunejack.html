Provably Fair & RTP

FortuneJack's provably fair dice game is one of the platform's most distinctive features — and one of its oldest. The dice game uses a SHA-256 hash-based system: the server seed hash is disclosed before each roll, and after the roll completes, both the server seed and client seed are revealed so the player can verify the outcome independently. The house edge on the dice game is configurable by the player, from 0% (break-even, used to verify the system works) up to a maximum house edge — most players use the 1% default setting.

Other FortuneJack originals (Plinko and Wheel variants) also run on provably fair mechanics using the same SHA-256 system. For the third-party slot library, RTPs follow provider standards — BetSoft titles average around 95.5%, Endorphina around 95%–96%, and Pragmatic Play at 96%+. FortuneJack holds a Curaçao licence and all software providers are certified by independent testing labs. The platform does not publish a site-wide RTP report, but individual RTPs are accessible within each game's information panel.
